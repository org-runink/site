How we charge for high-performance execution.

Core gives you the eyes ($20/mo). Cockpit gives you the hands ($80/mo). If you just want to see problems, buy Core. If you want us to fix them, buy Cockpit.
We charge separately for Data Units (DU) and Compute Units (CU). Each costs $0.01 per unit (rounded up):
• 1 DU = 1GB of data processed
• 1 CU = 60 seconds of compute time
These are direct pass-through costs from cloud providers with zero markup.
Transparency. We run on multiple cloud providers (AWS, GCP, Azure). Each has different infrastructure costs. By separating DU and CU, you see exactly what you’re paying for: data movement vs. actual computation. No hidden fees, no inflated margins.
